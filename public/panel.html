Sobre "modificar usuarios": los logins del panel (vos y cualquier otro operador) se manejan desde Supabase → Authentication → Users, no desde acá. Es intencional: crear o editar usuarios requiere una clave (service_role) que nunca debe exponerse en una página web pública — cualquiera que abriera el código del navegador podría usarla para borrar toda la base de datos. Si más adelante querés un botón dentro del panel para esto, se puede construir con una función de servidor separada (Netlify Function) que guarde esa clave de forma segura — avisame cuando lo necesites.
